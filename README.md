# CSV Profiler

A simple Python project that creates summary reports for CSV files, including basic statistics, missing values, and column types.

## Features

- Generate JSON and Markdown summary reports from any CSV file.
- Display basic statistics such as total rows, columns, missing values, and data types.
- Use a simple command-line interface (CLI) for quick and easy profiling.
- Launch an optional Streamlit web app to explore CSV summaries interactively.

## Installation

### Prerequisites
- Python 3.10 or higher
- UV installed
